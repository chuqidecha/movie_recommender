# 在MovieLens 1M数据集上使用深度学习进行评分预测和推荐

1. python3.6.7
2. tensorflow1.10.0

1. pre-processing.py 数据预处理模块
2. dataset.py 数据集 
3. inference.py  网络实现
4. train.py 网络训练
5. test.py 模型测试
6. features.py 使用训练好的模型提取特征
7. predicted.py 评分预测和相似度计算
